# Heart Disease Prediction

## Objective
Analyze clinical data to predict the presence of heart disease and identify key risk factors.

## Dataset
UCI Heart Disease Dataset (Cleveland)

## Project Structure
- Data/ Raw dataset
- Notebooks/EDA and analysis

### Day 2

### Day 3
